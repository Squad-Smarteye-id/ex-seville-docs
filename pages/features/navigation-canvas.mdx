# Navigation Canvas

- **Overview**: The Navigation Canvas is a feature that enables navigation through different areas or scenes based on their respective area or scene index.
- **Adding This Feature**: You can add this feature by clicking the ‘Add Features’ button found in the ‘EnvAreaHandler’ (VR360) or ‘EnvironmentAreaManagerVR3D’ (VR3D) component in the inspector and selecting the ‘Navigation Canvas’ prefab.
- **Setup:**
    - Input the index of the desired target area in the 'Target Next Scene or Area' variable.

# Seville Package v2.1

**Feature Updates:**

- **Quest System:** Enhanced to include standalone quests that don't require integration with other systems.
- **(New) Head Canvas:** Developers can now display notification messages to players using functions available in the 'HeadCanvasController' component.

**Bug Fixes:**

- **Quest System:** Resolved an issue with the Unity Event 'OnQuestFinished' that was not triggering as intended.

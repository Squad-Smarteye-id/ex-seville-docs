To change the target platform to Android in Build Settings, go to File Menu > Build Settings, select Android, and click 'Switch Platform'.

### **Setup Player Settings**

In Unity's Project Settings for the player:

- **Color Space:** Set to 'Linear' for accurate lighting and shading.
- **Texture Compression Format:** Choose 'ASTC' for high-quality, efficient texture compression.
- **Minimum API Level:** Set to 'Android 6.0 'Marshmallow' (API level 23)' for compatibility.
- **Scripting Backend:** Select 'IL2CPP' for performance and C++ integration.
- **Target Architectures:** Opt for 'ARMv7' or 'ARM64' based on device compatibility.
